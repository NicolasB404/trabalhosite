Seja bem-vindo

Este é um site feito para ajudar a população a não se endividar. Só o que você precisa fazer é informar quais são os seus gastos fixos mensais, logo após isso uma calculadora automática irá calcular a sua despesa total, para tornar mais fácil de controlar as suas despesas, de forma a não ficar endividado. Só o que você precisa fazer é clicar no botão a baixo.
